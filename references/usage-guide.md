# 使用指南

## 本技能做什么

本技能帮助 ChatGPT 从产品信息出发，创建工业级电商详情页生成工作流。它面向中文电商和 AI 出图流程，适合需要的不只是单条提示词，而是一整套可执行、可质检、可复用生产方案的用户。

可输出：

- 结构化产品 brief
- 类目与合规画像
- 逐页详情页规划
- 移动端电商文案
- 统一视觉系统
- 逐页图像生成提示词
- 后期叠字排版规范
- 质检清单和失败页重试说明

## 最佳使用场景

适用于：

- 抖音电商竖版详情页
- 淘宝 / 天猫商品详情图
- 小红书商品种草图
- 拼多多商品销售视觉
- 电商商品 AI 出图提示词包
- 工业化商品详情页提示词流程
- 审核现有详情页提示词是否可批量生产

## 推荐用户输入

高质量输入包含：

```text
产品名称：
产品类目：
产品简介：
核心卖点：
目标人群：
使用场景：
平台：
比例：9:16
图片数量：8张
风格：
主色调：
是否有原图：
必须出现：
禁止出现：
文案要求：自动生成 / 我提供文案
补充要求：
```

## 最小可运行输入

即使输入很少，也可以运行，例如：

```text
产品：黄芪
平台：抖音电商
比例：竖版
目标人群：30岁以上宝妈
风格：户外背景、暖黄配色、扁平风
文案：自动生成
输出：8张连贯详情页图像提示词
```

字段缺失时，应保守推断并标记为假设。

## 推荐请求方式

完整工业化工作流：

```text
请根据以下产品信息，输出完整工业级电商详情页生成包：Brief、合规边界、页面结构、逐页文案、统一视觉系统、逐页图像提示词、排版规范、质检与失败页重试方案。
```

仅要图像提示词：

```text
请只输出逐页图像生成提示词和 negative prompt，要求适合后期叠加中文文案。
```

审核现有提示词：

```text
请评估这份详情页提示词是否能用于工业级批量生成，并给出改造方案。
```

## 推荐生产流程

1. 用本技能生成结构化 brief 和策略。
2. 审核合规边界与文案。
3. 逐页生成图片背景或页面视觉。
4. 用设计/排版工具叠加精确文字。
5. 进行 OCR、视觉和合规质检。
6. 只重试失败页面。

## 重要限制

- 不要依赖图像模型准确渲染大段中文。
- 不要生成无证据强宣称或虚假资质。
- 高风险类目商用前仍需人工复核。
- 最终平台合规取决于卖家资质、产品类目和平台最新规则。

## V2 推荐补充字段

如有条件，加入以下信息可提高转化规划质量：

```text
流量来源：搜索 / 短视频 / 直播 / 推荐流 / 小红书种草 / 广告 / 私域
用户阶段：冷启动 / 已有需求 / 正在比价 / 准备下单
客单价：低 / 中 / 高
是否标品：是 / 否
已有证据素材：检测报告 / 买家秀 / 真实评价 / 资质 / 参数表 / 无
平台：淘宝 / 天猫 / 拼多多 / 抖音 / 小红书 / Amazon / Shopify
```

## V2 生产流程

1. 生成结构化 brief。
2. 生成平台与流量适配。
3. 生成买家决策路径。
4. 生成卖点-证据映射。
5. 生成合规画像。
6. 生成页面策略。
7. 生成文案。
8. 生成视觉资产规划。
9. 生成视觉系统和逐页提示词。
10. 在设计系统中叠加精确文字。
11. 执行转化、合规、视觉、OCR 和排版质检。
12. 除非策略本身错误，否则只重试失败页。
