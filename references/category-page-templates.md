# 类目页面模板

按产品类目选择模板，再根据用户 brief 调整页数和重点。

## 通用转化逻辑

1. attention：停止滚动，让产品身份清楚
2. understanding：说明产品是什么、为什么重要
3. desire：创造场景、质感、生活方式或结果期待
4. trust：展示细节、规格、材质、流程或安全使用逻辑
5. action：用购买理由和下一步收口

## 通用 8 页模板

1. **封面主视觉**：产品身份、情绪钩子、主利益
2. **核心卖点**：3-5 个易扫读卖点
3. **细节/材质/原料**：质感、材质、成分或工艺
4. **场景/人群共鸣**：目标用户和使用时刻
5. **使用方式/功能说明**：如何使用、穿着、涂抹、食用、清洁或安装
6. **优势/选择理由**：为什么值得选
7. **规格/包装/参数**：理性信息和 SKU 清晰化
8. **信任/成交收口**：总结、家庭/日常使用、购买意图

## 食品 / 饮料 / 滋补原料

推荐序列：

1. 封面主视觉：产品 + 自然场景 + 温暖日常饮用钩子
2. 核心卖点：自然原材、易冲泡、日常搭配、干净外观
3. 原材品质：纹理、切片/颗粒/色泽/洁净度特写
4. 日常饮用场景：居家、办公、户外休息、家庭餐桌
5. 冲泡/搭配方式：杯子、保温杯、茶壶、简单配方
6. 细节展示：产品形态、开袋/开盒、份量建议（如已知）
7. 包装规格：包装、数量、储存提醒（如提供）
8. 家庭常备/收口：温暖生活方式购买理由

安全角度：生活方式、自然原材、日常仪式感、便利。避免医疗/健康功能宣称。

## 护肤 / 个护

推荐序列：

1. 封面主视觉：产品质地和干净美妆氛围
2. 痛点共鸣：日常使用不适或需求，不制造医疗恐惧
3. 核心卖点：质地、体验、便利、日常护理匹配
4. 成分/配方亮点：仅使用用户提供成分或一般质感线索
5. 使用肤感/场景：浴室、梳妆台、旅行、日常流程
6. 使用步骤：清晰步骤和用量（如已知）
7. 包装规格：瓶/罐/管规格、便携性、卫生感
8. 收口成交：日常护理完整感和情绪价值

## 家居 / 清洁

推荐序列：

1. 封面主视觉：干净家庭改造氛围
2. 痛点切入：日常脏乱或不便
3. 核心功能：清洁、收纳、便利、设计
4. 场景适配：厨房、浴室、客厅、卧室
5. 使用步骤：简单 before/during/after 流程
6. 对比优势：便利和体验，不写无证据绝对宣称
7. 细节规格：材质、尺寸、包装、替换装、兼容性
8. 收口成交：家庭升级和日常轻松感

## 服饰

推荐序列：

1. 封面主视觉：模特/产品氛围和风格身份
2. 上身/搭配氛围：目标场景和时尚情绪
3. 面料细节：纹理、垂坠、透气性仅在提供时写
4. 版型亮点：剪裁、廓形、细节
5. 多场景穿搭：通勤、休闲、旅行、约会、居家
6. 颜色/尺码：SKU 和尺码指引
7. 工艺细节：走线、纽扣、腰部、口袋、拉链
8. 收口成交：衣橱价值

## 电子 / 配件

推荐序列：

1. 封面主视觉：产品和核心使用场景
2. 核心功能：用户提供的 3-5 个关键功能或参数
3. 痛点解决：线缆杂乱、电量焦虑、兼容性、便携
4. 参数亮点：功率、尺寸、材质、接口、容量（如提供）
5. 场景适配：办公、旅行、车载、居家、游戏
6. 兼容说明：已知设备和限制
7. 细节展示：接口、质感、按键、包装清单
8. 收口成交：日常实用升级

## 类目转化公式

### 服饰公式

合身/风格焦虑 → 上身或廓形证明 → 面料与舒适细节 → 搭配场景 → 颜色/尺码决策 → 工艺细节 → 护理/服务 → 衣橱价值。

风险控制：不要承诺显瘦、身材改变或万能合身；不要虚构面料成分、模特身高体重或耐洗性。

### 食品 / 饮料公式

食欲或日常仪式感 → 原料/形态可见证明 → 口感/质地预期 → 食用/饮用场景 → 包装/份量清晰 → 储存/物流 → 家庭常备收口。

风险控制：不要写治疗、改善身体机能、免疫、睡眠、疲劳、肝肾、血、祛湿、上火或药品化宣称。未提供产地、等级、年份、有机或检测，不要虚构。

### 护肤 / 个护公式

日常护理需求 → 质地/肤感证明 → 用户提供成分或配方亮点 → 使用步骤 → 日常场景 → 包装卫生/便携 → 日常护理收口。

风险控制：不要承诺治疗、治愈、保证修复、永久结果、100% 不过敏、医用级效果或治愈式前后对比。

### 家居 / 清洁公式

家庭脏乱或不便 → 产品解决方案 → 功能证明 → 房间场景 → 使用步骤 → 材质/尺寸/包装 → 服务/替换装 → 日常家庭升级收口。

风险控制：没有证据时不要写 100% 杀菌、永久防霉、疾病预防或健康保护。

### 电子 / 配件公式

使用痛点 → 核心规格/功能 → 兼容边界 → 场景证明 → 接口/细节镜头 → 安全/耐用证据（如提供） → 包装清单 → 日常实用升级收口。

风险控制：不要虚构功率、容量、认证、兼容性、芯片、材质、防水等级或电池表现。

### 美容工具 / 小家电公式

使用问题 → 产品机制 → 模式/功能说明 → 细节/配件证明 → 日常场景 → 清洁/收纳 → 规格/安全信息（如提供） → 便利收口。

风险控制：不要暗示医疗、塑形、生发、皮肤治疗或保证结果，除非明确支持且合法。

### 宠物产品公式

宠物主人问题 → 宠物友好设计 → 材质/尺寸证明 → 使用场景 → 清洁维护 → 体型/品种适用 → 包装/服务 → 日常照护收口。

风险控制：不要承诺医疗、行为治愈、绝对安全或兽医背书。

### 礼品公式

场合情绪 → 产品意义 → 包装呈现 → 个性化（如提供） → 使用或陈列场景 → 收礼人匹配 → 配送/服务 → 送礼收口。

风险控制：不要虚构奢侈身份、官方联名、知名设计师或文化认证。

## 页数逻辑

- 5-6 页：简单、低价、冲动型产品；合并证明与细节、合并规格与收口。
- 7-8 页：多数移动电商产品默认。
- 9-10 页：高价、高风险、多 SKU、技术型、服装尺码或需要更多证明的产品。
- 11 页以上：仅在用户明确要求长页，或产品确有证据、FAQ、对比和 SKU 复杂性时使用。

## 高转化页面序列模板

### 6 页直效模板

1. 主视觉和核心利益
2. 问题或使用相关性
3. Top 3 卖点
4. 可见证明或细节
5. 规格/包装/使用清晰化
6. 优惠/服务/行动收口

### 8 页平衡模板

1. 主视觉和点击验证
2. 买家相关性或痛点
3. 核心价值点
4. 细节/材质/规格证明
5. 使用或生活场景
6. 差异化或选择理由
7. SKU/规格/风险降低
8. 信任/行动收口

### 10 页考虑型购买模板

1. 主视觉和核心承诺
2. 买家问题和相关性
3. 解决方案概览
4. 功能-利益拆解
5. 证据/细节证明
6. 使用流程或场景
7. 对比或选择指南
8. 规格/SKU/兼容性
9. 信任/服务/FAQ
10. 优惠/行动收口
