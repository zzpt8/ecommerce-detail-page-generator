# 电商详情页合规规则

这些规则是实用生成约束，不构成法律意见。使用它们可以降低电商文案和视觉生成风险。

## 通用限制

不要虚构或暗示：

- 认证、检测、测试报告、专利、奖项、许可证或官方批准
- 销量、排名、市场份额、复购率或客户评价事实
- 明星、医生、机构、专家、平台或政府背书
- 精确产地、生产工艺、等级、年份、批次或供应商事实，除非用户提供
- “全网第一”“最有效”“永久”“100%”“保证”等绝对化优势

除非用户提供证据且类目允许，否则避免：

- 疾病预防或治疗
- 生理功能改善
- 保证结果
- 前后对比式变化宣称
- 攻击式竞品对比
- 暗示不购买会造成伤害的恐惧型紧迫感

## 高风险类目

以下类目使用严格模式：

- 食品、饮料、滋补原料、茶、营养、膳食产品
- 健康、养生、老人、孕产妇、婴幼儿、儿童
- 护肤、个护、护发、口腔护理
- 带抗菌/消毒宣称的清洁产品
- 医疗器械、药品、康复、止痛、助眠、减重

## 食品 / 饮料 / 滋补原料规则

避免：

- 治疗、预防、改善、调理、修复身体
- 增强免疫力、提高抵抗力
- 补气、养血、祛湿、降火、护肝、养肾
- 缓解疲劳、改善睡眠、改善体虚
- 将“药食同源”作为卖点证明，除非用户提供合规上下文
- 老少皆宜、孕妇可用、儿童可用，除非用户提供资质

更安全方向：

- 日常泡饮
- 自然原材
- 温暖陪伴
- 轻松搭配
- 家庭常备
- 日常饮养氛围
- 一杯热饮的生活仪式感
- 适合居家、办公、户外小憩等场景

风险到安全替代表达：

| 风险表达 | 安全替代 |
| --- | --- |
| 补气养血 | 日常温和饮养 |
| 增强免疫力 | 给日常生活一点温暖补给 |
| 改善疲劳 | 忙碌之后的一杯暖饮陪伴 |
| 调理身体 | 轻松融入日常饮用习惯 |
| 体虚人群适用 | 适合喜欢日常饮养的人群 |
| 药材级品质 | 原材质感看得见 |
| 功效显著 | 体验更安心、更日常 |

## 护肤和个护规则

避免：

- 根治、治疗、医美级疗效、药用疗效
- 永久祛斑、彻底祛痘、保证修复皮肤屏障
- 100% 不过敏、孕妇绝对安全
- 暗示医学治愈的前后对比图

更安全方向：

- 清爽肤感
- 温和护理
- 日常清洁
- 细腻质地
- 使用体验
- 肤感舒适
- 清洁更省心

## 母婴规则

避免：

- 绝对安全、零风险、医生推荐，除非用户提供且合法
- 治疗湿疹、改善发育、提升智力
- 夸张恐惧话术

更安全方向：

- 柔软亲肤
- 日常照护
- 细致呵护
- 使用更省心
- 家庭场景友好

## 清洁产品规则

避免：

- 100% 杀菌、永久防霉、彻底除菌，除非有充分证据
- 超出清洁用途的健康保护宣称

更安全方向：

- 日常清洁更方便
- 家庭打理更省心
- 清爽洁净感
- 适合厨房、浴室、客厅等场景

## 电子电器规则

避免：

- 虚假参数、虚假兼容性、虚假认证
- 永久耐用承诺
- 无证据性能排名

更安全方向：

- 使用更便捷
- 连接更顺手
- 体验更稳定
- 适配常见场景
- 参数以用户提供信息为准

## 视觉限制

避免：

- 普通食品/养生产品出现医院、医生、药瓶、医疗十字、诊断图、器官、疾病或疼痛视觉
- 假认证徽章、检测印章、奖牌、排名皇冠
- 极端前后对比
- 未授权平台 logo、明星脸、知名品牌包装、水印
- 用户未提供证据时，暗示儿童、孕妇、老人可用的视觉

## 合规输出规则

中高风险类目必须包含合规提示，例如：

“本方案采用生活方式与日常使用表达，已规避治疗、改善身体机能、绝对化承诺和虚构资质类表述。”

## 证据驱动合规层

合规不只是禁词，还取决于主张是否有足够支持。先使用 `claim-evidence-rules.md`。

### 主张强度门槛

- 强主张必须有用户明确提供的证据。
- 中等主张需要用户提供事实、可见产品证据或谨慎措辞。
- 软主张可以来自场景和体验，但不能暗示保证结果。

### 必须要求用户证据的内容

- 认证、检测、测试报告、专利、奖项、许可证、官方批准
- 销量、排名、市场份额、复购率、用户评价事实
- 明星、医生、机构、专家、平台或政府背书
- 精确产地、生产工艺、等级、年份、批次、供应商事实
- 医疗、健康、生理、消毒、抗菌、防水、安全、兼容性、耐用性或具体可测性能主张

### 安全输出规则

如果某个主张有吸引力但无证据：

1. 不写成面向客户的事实。
2. 加入缺失证据。
3. 提供软性安全替代。
4. 用细节、场景或生活方式语言替代证明语言。

### 视觉合规规则

不要创造用户没有提供的视觉证明。特别不要生成：

- 证书卡、检测印章、官方章、奖牌徽章
- 普通消费品的医生/医院/实验室场景
- 假买家评价截图或假聊天截图
- 排名皇冠、平台 logo 背书、政府样式徽章
- 未证实功效的前后变化视觉
