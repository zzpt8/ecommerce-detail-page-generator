# Ecommerce Detail Page Generator 

一个面向中文电商场景的工业级详情页生成 Skill，用于把产品信息转化为可执行的详情页策划、文案、图像提示词、排版规范、合规边界、质检与失败页重试方案。

它不是简单生成一条图片提示词，而是按“用户决策路径 + 卖点证据链 + 平台适配 + 视觉资产规划 + 合规质检”的方式，输出可供运营、设计师、开发者或图像生成流水线使用的完整生产包。

## 适合什么场景

适合用于：

- 淘宝 / 天猫商品详情页
- 拼多多商品卖点页
- 抖音电商竖版详情页或直播/短视频承接页
- 小红书种草型产品页
- Amazon A+ / 跨境商品图文模块
- AI 生图工作流中的逐页提示词生产
- 已有详情页 prompt 或页面方案的工业化审核
- 批量商品详情页生成流程设计

##  核心升级

V相比基础详情页生成流程，新增了 5 个关键模块：

1. **平台与流量适配**  
   根据淘宝、天猫、拼多多、抖音、小红书、Amazon A+、DTC 等不同平台，调整页面密度、文案语气、信任模块、视觉风格和成交收口方式。

2. **用户决策路径图**  
   不再按产品功能机械排页，而是先判断用户每一屏想确认什么：是不是我要的、有什么用、凭什么信、会不会买错、为什么现在买。

3. **卖点-证据-合规映射**  
   每个强卖点都必须绑定用户提供证据、可见产品证据或降级为安全表达，避免虚构检测、资质、销量、功效、产地、权威背书等高风险内容。

4. **视觉资产规划**  
   在生成逐页图像 prompt 前，先规划主视觉、细节特写、场景图、规格图、包装图、步骤图、对比图、信任证明图等镜头资产，避免每一页都变成同质化海报。

5. **转化质量 QC**  
   质检不只看图片好不好看，还会检查首屏清晰度、一屏一卖点、前 3 页转化信息、证据链完整性、平台风格匹配度、移动端可读性和失败页重试方式。

## 最小可运行输入

即使只有少量信息，也可以启动：

```text
产品：便携榨汁杯
平台：抖音电商
目标人群：上班族和健身人群
风格：清爽、年轻、明亮、竖版 9:16
输出：8 张详情页方案和逐页图像生成提示词，文案自动生成
```

## 推荐提问方式

### 生成完整详情页生产包

```text
请根据以下产品信息，输出完整工业级电商详情页生成包：Brief、平台适配、用户决策路径、卖点证据链、合规边界、页面结构、逐页文案、视觉资产规划、统一视觉系统、逐页图像提示词、排版规范、质检与失败页重试方案。

[粘贴产品信息]
```

### 只要生图提示词

```text
请只输出逐页图像生成提示词和 negative prompt，要求适合后期叠加中文文案，并保持 8 张图视觉一致。

[粘贴产品信息]
```

### 审核已有详情页方案

```text
请审核下面这套详情页方案是否适合工业级批量生成，重点检查转化逻辑、卖点证据链、平台适配、合规风险、视觉一致性和失败页重试策略。

[粘贴现有方案或 prompt]
```

### 为设计师输出排版说明

```text
请基于这个产品生成详情页排版规范：每页文字层级、文本位置、安全边距、产品区域、不可遮挡区域、移动端可读性和后期叠字说明。

[粘贴产品信息]
```

完整模式下，Skill 通常会输出：

1. 项目概览
2. 结构化产品 Brief
3. 平台与流量适配判断
4. 用户决策路径图
5. 卖点-证据-合规映射表
6. 合规边界
7. 详情页页面规划
8. 逐页文案
9. 视觉资产规划
10. 统一视觉系统
11. 逐页图像生成提示词
12. 排版规范、质检清单与失败页重试方案

## 生产使用建议

推荐工作流：

1. 先用 Skill 生成完整策划包。
2. 人工确认产品事实、证据材料和合规边界。
3. 按“逐页图像生成提示词”生成每一页底图。
4. 不要依赖图像模型生成大段中文，建议使用排版工具后期叠字。
5. 按 layout spec 叠加准确文案。
6. 用 QC 清单检查页面目标、产品一致性、文字准确性、视觉统一和合规风险。
7. 只重试失败页面，不要轻易推翻整套视觉系统。

## 重要限制

- 不会虚构认证、检测报告、销量、产地、奖项、专家背书或用户评价。
- 对食品、滋补、营养、护肤、母婴、老人、儿童、医疗相关产品会使用更严格的安全表达。
- 不建议让图像模型直接生成大量中文文案，最终商用图应以后期排版文字为准。
- 高风险类目上线前仍需人工复核平台规则、广告法要求和商家自身资质。
